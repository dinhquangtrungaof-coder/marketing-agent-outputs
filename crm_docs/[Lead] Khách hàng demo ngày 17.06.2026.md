## 1. LEAD CLASSIFICATION

```
┌─────────────────────────────────────────────────────┐
│  PHÂN LOẠI: ☀️ ẤM (WARM LEAD)                      │
│  Lý do: Có intent rõ nhưng thiếu thông tin cụ thể  │
│  Chưa nêu: số lượng, timeline, sản phẩm cụ thể     │
└─────────────────────────────────────────────────────┘
```

| Signal | Đánh giá | Điểm |
|--------|----------|------|
| Chủ động liên hệ | ✅ Có | +2 |
| Nêu cụ thể sản phẩm | ⚠️ Chỉ nói "bao bì" chung | +1 |
| Số lượng / timeline | ❌ Chưa có | 0 |
| Đã có file thiết kế | ❓ Chưa biết | 0 |
| Ngân sách đề cập | ❌ Chưa có | 0 |

---

## 2. LEAD SCORING

```
TỔNG ĐIỂM HIỆN TẠI: 3/10
━━━━━━━━━━━━━━━━━━━━━━━━
0–3  = Lạnh → Nuôi content
4–6  = Ấm  → Inbox ngay, hỏi lọc
7–10 = Nóng → Gọi điện trong 30 phút
━━━━━━━━━━━━━━━━━━━━━━━━
⭐ Lead này = 3 điểm → Ranh giới Lạnh/Ấm
→ XỬ LÝ: Inbox ngay để đẩy lên 6+ điểm
```

**Bộ câu hỏi lọc để tăng điểm:**

| Câu hỏi | Nếu trả lời tốt | +Điểm |
|---------|-----------------|-------|
| Bao bì loại nào? (hộp/tem/túi) | Nêu cụ thể | +1 |
| Số lượng cần in? | ≥ 500 cái | +2 |
| Đã có file thiết kế chưa? | Có file AI/PDF | +2 |
| Cần trong bao lâu? | Dưới 2 tuần | +2 |

## 3. INBOX SCRIPT

### 📨 TIN NHẮN MỞ ĐẦU *(Gửi trong 15 phút)*

```
Chào bạn! 👋

Bên mình là In ấn Thành Trung —
chuyên in bao bì theo file có sẵn của khách.

Bạn đang cần in loại bao bì nào ạ?
  □ Hộp giấy
  □ Tem dán / nhãn sản phẩm
  □ Túi giấy kraft
  □ Combo cả bộ

Hỏi để báo giá đúng ngay cho bạn luôn 🙌
```

> 💡 **Lý do:** Câu hỏi trắc nghiệm → dễ trả lời → tỷ lệ phản hồi cao hơn câu hỏi mở

---

### 📨 CÂU HỎI LỌC *(Sau khi khách trả lời — gửi 1 tin, hỏi tối đa 2 điều)*

**Nếu khách chọn Hộp giấy:**
```
Hộp giấy bên mình in từ 200 cái, 
đơn giá từ 4.000đ/hộp nhiều màu 🎨

Bạn cần khoảng bao nhiêu cái và 
đã có file AI/PDF chưa ạ?

(Có file → mình báo giá cố định trong 30 phút luôn)
```

**Nếu khách chọn Tem dán:**
```
Tem dán chống nước bên mình từ 100 tem,
giá 180đ/tem (18k/100 tem) 🏷️

Bạn cần số lượng khoảng bao nhiêu
và tem in mấy màu ạ?
```

**Nếu khách chọn Combo cả bộ:**
```
Combo Hộp + Tem + Túi đồng bộ là thế mạnh 
của bên mình — 1 đầu mối, nhận hàng 1 lần 📦

Bạn đang ra mắt sản phẩm gì ạ?
Mình tư vấn combo phù hợp và báo giá trọn gói luôn.
```
